# Chicken Jockey
Annoy your friends and GMs in WoW 1.12.  Create altered item names and rarities from legitimate items, and share them in chat.  

### Installation
1 - Download the zip file  
2 - Extract the contents  
3 - Rename the directory ChickenJockey (remove '-main')  
4 - Copy directory and its contents to WoW\Interface\Addons\  

### Usage
1 - Open with /cj, /chickenjockey or the minimap button  
2 - Enter an item id you found at https://database.turtlecraft.gg  
2a - Example: https://database.turtlecraft.gg/?item=12544, the item id is 12544  
3 - Click 'Fetch'  
3a - If it is a valid item id, the title and rarity will populate  
3c - Alternatively, use 'Random' to get a random item  
4 - Modify the item name and rarity  
5 - Click 'Generate Link'  
6 - A link is sent to your main chat window that can now be shared by shift clicking
